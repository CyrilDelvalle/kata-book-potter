# kata-book-potter

This project is the coding dojo kata potter.

## Dependancies

This project use NodeJs, Express and React.

## Run Project

To run the Api server, go to the back folder and run

### `yarn start`

or

### `npm run start`

You can send request on [http://localhost:3001](http://localhost:3001)

To run all unit tests run

### `yarn test`

or

### `npm run test`

To run the front, go to the front folder and run

### `yarn start`

or

### `npm run start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
